# Unit Converter

### Concepts implemented : 

- primitive datatypes ```char, int, float```
- comparison operator ``` == ```
- Arithmetic operators ``` +, -, *, /, % ```
- Expressions eg. ``` fahrenheitToCelcius =  ((userinputF-32) * (5.0/9.0)); ```
- ```if-else``` ladder
- ```nested if-else``` ladder

### Output :

- Fahrenheit to Celsius
- Celsius to Fahrenheit
